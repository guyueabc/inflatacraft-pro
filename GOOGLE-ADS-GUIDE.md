## 第一步: 创建广告系列

```
1. ads.google.com → 广告系列 → + 新建广告系列
2. 目标: 潜在客户 (Leads)
3. 类型: 搜索广告 (Search)
4. 命名: "Search | InflatableModel"
5. 预算: $20/天
6. 出价: 点击次数 → 设置最高每次点击费用出价上限
7. 网络: 仅 Google 搜索 (取消搜索合作伙伴和展示网络)
8. 地区: United States, UK, Australia, Canada, Germany, UAE
9. 语言: English
10. 时段: 8:00-22:00 (目标市场时间)
```

## 第四步: 转化追踪

```

2. 选择: 网站
3. 转化名称: Quote Form Submit
4. 类别: 提交潜在客户表单
```
